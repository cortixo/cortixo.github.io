A Free Javascript Library


Learn-Rate:

Cortixo Commands

cortixo.synapses() - total synapse count
cortixo.neurons() - total neuron count
cortixo.learnRate(number) - set learn rate (0-2)
cortixo.reset() - reset network cortixo.inputLayer(number) - create input layer
cortixo.hiddenLayer(number) - create hidden layer
cortixo.outputLayer(number) - create output layer
cortixo.setInput(array) - set input neurons
cortixo.expectedOutput(array) - what the answer should be (put before train)
cortixo.train() - train cortixo (one iteration)
cortixo.run() - run without training
cortixo.output() - outputs array of output layer


How do I use this?

This plug-in is still in development to activate it on your website however use: 

<script type="application/javascript" src="https://cortixo.github.io/src/ai.js"> </script>
And to show the loss chart this can be used: 
<canvas id="cortixoUI" width=300px height=300px> </canvas>
